التمرين الثاني (Exercice 2 - 6 نقاط) - Répondre en langage algorithmique

التمرين هذا يحكي على زوج مفاهيم: الأعداد المتعاكسة (opposés) والأعداد المقلوبة (inverses).

  • الأعداد المتعاكسة: هما عددان المجموع نتاعهم يساوي صفر (0). مثلاً: 5 و -5.
  • الأعداد المقلوبة: هما عددان الجداء (الضرب) نتاعهم يساوي واحد (1). مثلاً: 4 و 1/4 (يعني 0.25).

أ) دالة التحقق من عددين (opposés ou inverses)

يطلب منك تكتب دالة (Fonction) ترجع Vrai (صحيح) إذا كان العددان اللي تمدّهملها (x و y) متعاكسين أو مقلوبين، وترجع Faux (خطأ) إذا ماكانوش. 

// الخوارزمية بالفرنسية
Fonction sontOpposesOuInverses (x : Reel, y : Reel) : Booléen
// Description : Retourne Vrai si x et y sont opposés OU inverses, Faux sinon.
// Données : x, y - les deux nombres à vérifier.
// Résultat : Vrai ou Faux.
Variables
    // Aucune variable locale nécessaire
Début
    Si (x + y = 0) Ou (x * y = 1) Alors
        Retourner Vrai
    Sinon
        Retourner Faux
    Fin Si
Fin

ب) دالة البحث في جدول (عن أي زوج)

عندك جدول (tableau) T فيه N أعداد صحيحة (Entiers). لازم تكتب دالة ترجع Vrai إذا لقيت على الأقل زوج أعداد في الجدول هذا (ماشي شرط يكونوا متتابعين) يكونوا متعاكسين أو مقلوبين. وترجع Faux إذا مالقيتش. هنا لازم تقارن كل عدد مع كامل الأعداد اللي موراه في الجدول. 

// الخوارزمية بالفرنسية
Fonction existeOpposesOuInverses (T : Tableau[1..N] d'Entiers, N : Entier) : Booléen
// Description : Retourne Vrai s'il existe deux éléments dans T (pas nécessairement consécutifs)
//               qui sont opposés ou inverses, Faux sinon.
// Données : T - le tableau d'entiers, N - la taille du tableau.
// Résultat : Vrai ou Faux.
Variables
    i, j : Entier
Début
    Pour i de 1 à N-1 Faire
        Pour j de i + 1 à N Faire
            // On utilise la fonction précédente.
            Si sontOpposesOuInverses(T[i], T[j]) Alors
                Retourner Vrai // On a trouvé une paire, on arrête.
            Fin Si
        Fin Pour
    Fin Pour
    Retourner Faux // Aucune paire trouvée après toutes les comparaisons.
Fin

ج) دالة البحث في جدول (عن زوج متتابع)

كيف كيف عندك جدول T. لازم تكتب دالة ترجع Vrai إذا لقيت فيه زوج أعداد متتابعين (يعني واحد مور خوه طول) يكونوا متعاكسين أو مقلوبين. وترجع Faux إذا مالقيتش. هنا تمشي على الجدول وتقارن كل عنصر بالعنصر اللي بعدو مباشرة. 

// الخوارزمية بالفرنسية
Fonction existeConsecutifsOpposesOuInverses (T : Tableau[1..N] d'Entiers, N : Entier) : Booléen
// Description : Retourne Vrai s'il existe deux éléments consécutifs dans T
//               qui sont opposés ou inverses, Faux sinon.
// Données : T - le tableau d'entiers, N - la taille du tableau.
// Résultat : Vrai ou Faux.
Variables
    i : Entier
Début
    // On vérifie les paires (T[1], T[2]), (T[2], T[3]), ..., (T[N-1], T[N])
    Si N < 2 Alors // Il faut au moins 2 éléments pour avoir une paire consécutive
        Retourner Faux
    Fin Si

    Pour i de 1 à N-1 Faire
        Si sontOpposesOuInverses(T[i], T[i+1]) Alors
            Retourner Vrai // On a trouvé une paire consécutive, on arrête.
        Fin Si
    Fin Pour
    Retourner Faux // Aucune paire consécutive trouvée.
Fin

التمرين الثالث (Exercice 3 - 6 نقاط) - Répondre en langage algorithmique

المطلوب في هذا التمرين هو العمل على جدول (tableau) اسمه T يحتوي على N أعداد صحيحة (Entiers). الهدف هو إيجاد قيمتين: أكبر قيمة (le plus grand) وثاني أكبر قيمة (le deuxième plus grand) في الجدول.

أ) نوع الإجراء (Action)، المعاملات (Paramètres) وطريقة الإرسال (Mode de transmission)

السؤال هو عن نوع الإجراء المبرمج (Action paramétrée) الأنسب: هل هو Fonction أم Procédure؟ ولماذا؟ وما هي المعاملات (paramètres) اللازمة وكيف يكون نمط الإرسال (mode de transmission) لكل منها؟

  • نوع الإجراء المفضل: **Procédure**.
  • السبب (Justification): نحتاج إلى إرجاع **قيمتين** مختلفتين (الأكبر والثاني الأكبر). الـ Fonction عادةً ترجع قيمة واحدة فقط من خلال اسمها. أما الـ Procédure، فهي لا ترجع قيمة باسمها ولكن يمكنها تعديل قيم المتغيرات المُمررة إليها كمعاملات إذا كان نمط الإرسال مناسبًا (مثل Résultat)، مما يسمح لها بإخراج عدة نتائج.
  • المعاملات (Paramètres) وأنماط الإرسال (Modes):
  • T : Tableau[1..N] d'Entiers (Mode **Donnée**) - الجدول المدخل، نقرأ منه فقط.
  • N : Entier (Mode **Donnée**) - حجم الجدول، نحتاج لمعرفته فقط.
  • max1 : Entier (Mode **Résultat**) - المتغير الذي ستكتب فيه الـ Procédure أكبر قيمة تجدها (نتيجة مُخرَجة).
  • max2 : Entier (Mode **Résultat**) - المتغير الذي ستكتب فيه الـ Procédure ثاني أكبر قيمة تجدها (نتيجة مُخرَجة أخرى).

(نمط Résultat هو الأنسب هنا لأن max1 و max2 لا يحملان قيمة إدخال مهمة للـ Procédure؛ بل هي التي ستقوم بحسابهما وتعيين قيمتهما).

ب) خوارزمية الحل (Procédure لإيجاد أكبر قيمتين)

المطلوب هنا هو كتابة الخوارزمية التفصيلية للـ Procédure التي تقوم بإيجاد أكبر قيمتين.

خطوات الخوارزمية:

  1. التحقق المبدئي: التأكد من أن حجم الجدول N هو 2 على الأقل. إذا كان أقل، لا يمكن إيجاد "ثاني أكبر قيمة"، ويجب التعامل مع هذه الحالة (مثل عرض رسالة خطأ أو الخروج).
  2. التهيئة (Initialisation): مقارنة أول عنصرين في الجدول (T[1] و T[2]). وضع الأكبر منهما في max1 والأصغر في max2 كقيم أولية.
  3. المسح (Parcours): إنشاء حلقة تكرار (Pour) تمر على باقي عناصر الجدول، بدءًا من العنصر الثالث (i من 3) حتى آخر عنصر (N).
  4. المقارنات والتحديث داخل الحلقة: لكل عنصر T[i]:
    • إذا كان T[i] > max1: هذا يعني أن T[i] هو الأكبر الجديد. نقوم بتحديث max2 ليأخذ القيمة القديمة لـ max1، ثم نحدّث max1 ليأخذ قيمة T[i].
    • إذا كان T[i] ليس أكبر من max1، نتحقق إذا كان T[i] > max2:
      • إذا كان كذلك، فهذا يعني أن T[i] هو ثاني أكبر قيمة جديدة (أو مساوية للأولى). نحدّث max2 ليأخذ قيمة T[i].
    • إذا لم يتحقق أي من الشرطين السابقين، ننتقل للعنصر التالي.
  5. النهاية: بعد انتهاء الحلقة، المتغيران max1 و max2 (اللذان تم تمريرهما بالنمط Résultat) سيحتويان على القيمتين المطلوبتين. 
// الخوارزمية بالفرنسية لـ Procédure
Procedure TrouverDeuxMax (T : Tableau[1..N] d'Entiers; N : Entier; Var max1, max2 : Entier)
// Description : Trouve le plus grand et le deuxième plus grand élément dans T.
// Précondition : N >= 2 (Le tableau doit avoir au moins 2 éléments).
// Données : T - le tableau, N - sa taille.
// Résultats : max1 - le plus grand, max2 - le deuxième plus grand (via paramètres Var).
Variables
    i : Entier
Début
    // Gérer le cas où N < 2
    Si N < 2 Alors
       Ecrire("Erreur : Le tableau doit contenir au moins 2 éléments.")
       Retourner // Sortir si la condition n'est pas remplie
    Fin Si

    // Initialisation avec les deux premiers éléments
    Si T[1] > T[2] Alors
        max1 <- T[1]
        max2 <- T[2]
    Sinon // T[2] >= T[1]
        max1 <- T[2]
        max2 <- T[1]
    Fin Si

    // Parcourir le reste du tableau à partir du 3ème élément
    Pour i de 3 à N Faire
        Si T[i] > max1 Alors
            // Nouvel élément est le plus grand
            max2 <- max1  // Ancien max1 devient le nouveau max2
            max1 <- T[i]  // T[i] est le nouveau max1
        Sinon // T[i] <= max1
            // Vérifier si T[i] est plus grand que le deuxième max actuel
            Si T[i] > max2 Alors
                max2 <- T[i] // T[i] devient le nouveau max2
            Fin Si
        Fin Si
    Fin Pour
// À la fin, max1 et max2 contiennent les résultats souhaités
Fin

ملاحظة: الخوارزمية أعلاه تعتبر أنه إذا كانت أكبر قيمتين متساويتين (مثل [5, 5, 3])، فإن النتيجة ستكون max1=5 و max2=5. إذا كان المطلوب البحث عن ثاني أكبر قيمة *مختلفة* عن الأولى، فيجب تعديل الشرط الثاني قليلاً ليصبح `Si T[i] > max2 Et T[i] < max1 Alors`.

ج) استخدام الـ Enregistrements (الكود الكامل للـ Fonction)

كما ذكرنا، يمكن استخدام Enregistrement لجمع النتيجتين (أكبر وثاني أكبر قيمة) في هيكل واحد. هذا يسمح لنا بكتابة Fonction بدلاً من Procédure، حيث تقوم الـ Fonction بإرجاع هذا الـ Enregistrement كقيمة وحيدة.

1. تعريف الـ Enregistrement (Déclaration du Type):

// أولاً، نحتاج تعريف النوع Enregistrement
Type
    ResultatDeuxMax = Enregistrement
        premier : Entier  // الحقل لتخزين أكبر قيمة
        deuxieme : Entier // الحقل لتخزين ثاني أكبر قيمة
    Fin Enregistrement

2. الكود الكامل للـ Fonction:

// الخوارزمية بالفرنسية للـ Fonction كاملة
Fonction TrouverDeuxMax_Enreg (T : Tableau[1..N] d'Entiers, N : Entier) : ResultatDeuxMax
// Description : تجد أكبر وثاني أكبر قيمة في T وترجعهم في Enregistrement واحد.
// Données : T - الجدول, N - حجمه.
// Précondition : N >= 2 (نفترض أن الشرط متحقق لتجنب تعقيد إدارة الأخطاء في هذا المثال)
// Résultat : قيمة من النوع ResultatDeuxMax تحتوي على القيمتين.
Variables
    i : Entier              // متغير للحلقة
    res : ResultatDeuxMax   // متغير من نوع Enregistrement لتخزين وإرجاع النتيجة
    // لا نحتاج لـ max1_local و max2_local، يمكن استخدام حقول 'res' مباشرة

Début
    // التحقق من الشرط المسبق (اختياري، لكن جيد):
    // إذا كان N < 2، يجب تحديد ما سيتم إرجاعه. هنا، سنفترض أن N>=2 دائمًا.
    // إذا أردنا التعامل مع الخطأ، يمكن إرجاع قيمة خاصة أو رفع خطأ.

    // التهيئة باستخدام أول عنصرين وتخزينها مباشرة في حقول المتغير 'res'
    Si T[1] > T[2] Alors
        res.premier <- T[1]
        res.deuxieme <- T[2]
    Sinon // T[2] >= T[1]
        res.premier <- T[2]
        res.deuxieme <- T[1]
    Fin Si

    // المرور على باقي عناصر الجدول من العنصر الثالث
    Pour i de 3 à N Faire
        Si T[i] > res.premier Alors
            // T[i] هو الأكبر الجديد
            res.deuxieme <- res.premier  // القيمة الأولى السابقة تصبح الثانية
            res.premier <- T[i]       // T[i] يصبح القيمة الأولى الجديدة
        Sinon // T[i] <= res.premier
            // التحقق إذا كان T[i] أكبر من القيمة الثانية الحالية
            Si T[i] > res.deuxieme Alors
                res.deuxieme <- T[i] // T[i] يصبح القيمة الثانية الجديدة
            Fin Si
        Fin Si
    Fin Pour

    // إرجاع الـ Enregistrement المحتوي على النتيجتين
    Retourner res
Fin

شرح الجسم (Corps) للـ Fonction:

  • التعريف: تبدأ بتعريف الـ Fonction واسمها، معاملاتها (T و N كـ Donnée)، ونوع القيمة المُرجعة (ResultatDeuxMax).
  • المتغيرات المحلية: نعرّف متغير للحلقة i، والأهم، متغير res من نوع ResultatDeuxMax. هذا المتغير سيُستخدم لتخزين النتيجتين وسيتم إرجاعه في النهاية.
  • التهيئة: نقارن T[1] و T[2] ونخزن الأكبر في res.premier والأصغر في res.deuxieme.
  • الحلقة والمقارنات: ندور على باقي العناصر (من 3 إلى N). داخل الحلقة، نقارن T[i] مع res.premier و res.deuxieme ونقوم بتحديث حقول res بنفس المنطق المستخدم في الـ Procédure.
  • الإرجاع (Retourner): في نهاية الـ Fonction، نستخدم التعليمة Retourner res لإرجاع المتغير res (الذي هو من نوع Enregistrement ويحتوي على القيمتين) كنتيجة وحيدة للـ Fonction.
